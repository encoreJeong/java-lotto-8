# 로또 게임 HTTP 기반 입출력으로 재구성
### [🎱 로또 시뮬레이터](https://kiasu-trace-rxpf.pagedrop.io/)<br><br>

## ⭐️ 주안점
> 3주차에서 구현한 클래스들을 최대한 활용하며, 구현했던 프로그램이 요구사항 변화에 유연한 구조인지 확인한다.
<br>

## ⚙️ 기능 요구사항

> - 콘솔 기반 입출력 로또 게임과 기능 [요구사항](https://github.com/encoreJeong/java-lotto-8)은 동일하다.
> - 콘솔 기반 입출력 인터페이스를 HTTP기반 으로 재구성한다.
> - HTTP 요청을 받아들이고 처리하는 로직은 [직접 만든 프레임워크](https://github.com/encoreJeong/encore-framework)를 이용한다.
<br>

## 💡 프로그래밍 요구사항
> - [Encore framework](https://github.com/encoreJeong/encore-framework) 를 기반으로 프로그램을 구현한다.
<br>
