## 개요

간단한 로또 발매기 구현

## ⚙️ 기능 요구사항

### 사용자 입력

1. 로또 구입 금액을 입력 받는다.
    1. 구입 금액 입력 형식이 유효한지 확인한다.
        1. 사용자 입력이 정수인지 확인한다.
    2. 구입 금액이 규칙을 만족하는지 확인한다.
        1. 로또를 최소 1개 이상 살 수 있는지 확인한다.
        2. 사용자 입력이 1,000으로 나누어 떨어지는지 확인한다.
    3. 유효하지 않으면 “[ERROR]” 로 시작하는 에러 메시지를 출력하고 그 부분부터 입력을 다시 받는다.


2. 당첨 번호를 입력 받는다.
    1. 당첨 번호 입력 형식이 유효한지 확인한다.
        1. 쉼표와 양의 정수로만 이루어진 문자열인지 확인한다.
    2. 당첨 번호의 규칙을 만족하는지 확인한다.
        1. 1부터 45 사이의 정수인지 확인한다.
        2. 중복되는 수가 없는지 확인한다.
        3. 정확히 6개의 수를 포함하고 있는지 확인한다.
    3. 유효하지 않으면 “[ERROR]” 로 시작하는 에러 메시지를 출력하고 그 부분부터 입력을 다시 받는다.


3. 보너스 번호를 입력 받는다.
    1. 보너스 번호의 입력 형식이 유효한지 확인한다.
        1. 정수인지 확인한다.
    2. 보너스 번호의 규칙을  만족하는지 확인한다.
        1. 1부터 45 사이의 정수인지 확인한다.
        2. 당첨 번호와 중복되지 않는지 확인한다.
    3. 유효하지 않으면 “[ERROR]” 로 시작하는 에러 메시지를 출력하고 그 부분부터 입력을 다시 받는다.

### 로또 발행

1. 발행가능한 로또의 수량을 구한다.


2. 발행가능한 로또의 수량만큼 로또를 발행한다.

### 당첨 내역 확인

1. 당첨된 로또들의 등수를 확인한다.
    - 당첨은 1등부터 5등까지 있다. 당첨 기준과 금액은 아래와 같다.
        - 1등: 6개 번호 일치 / 2,000,000,000원
        - 2등: 5개 번호 + 보너스 번호 일치 / 30,000,000원
        - 3등: 5개 번호 일치 / 1,500,000원
        - 4등: 4개 번호 일치 / 50,000원
        - 5등: 3개 번호 일치 / 5,000원


2. 당첨된 로또들의 총 당첨금액을 구한다.
   - 총 당첨금액의 최댓값은 4,294,966,000,000,000이다. (구입가능한 최대 로또 개수) * (1등 당첨금액) 


3. 수익률을 구한다.
    - 수익률은 총 당첨금액 / 로또 구입 금액이다.
    - 수익률은 소수점 둘째 자리에서 반올림한다. (ex. 100.0%, 51.5%, 1,000,000.0%)


### 출력

1. 발행된 로또 출력
    1. 발행가능한 로또의 수량을 출력한다.
    2. 발행된 로또 번호를 출력한다.


2. 당첨 내역 출력


3. 수익률 출력
